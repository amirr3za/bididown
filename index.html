BidiDown: Bidirectional Markdown editor.

L: Create a new LTR block.

R: Create a new RTL block.

T: Toggle, change the direction of current block.

D: Delete the current block.

S: Save, locally.

O: Open, open a markj document.

H: HTML, shows the result in a new seprate window.
